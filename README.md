# 📝 Lista de Tarefas

Projeto simples de lista de tarefas desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Salvar tarefas no navegador (localStorage)

## 💻 Tecnologias
- HTML
- CSS
- JavaScript

## 📌 Sobre
Este projeto foi desenvolvido como prática de Front-End.
